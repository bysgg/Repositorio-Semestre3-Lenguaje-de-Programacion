## 🛠️ Competencias Desarrolladas

Durante este periodo, se aplicaron los siguientes pilares de la ingeniería de software:

- **Abstracción y Encapsulamiento**: Uso de clases ABC y atributos privados para proteger la lógica de negocio.  
- **Herencia y Polimorfismo**: Creación de jerarquías escalables (Cuadros/Esculturas) y métodos sobreescritos.  
- **Principios SOLID**: Especialmente Responsabilidad Única e Inversión de Dependencias.  
- **Modelado UML**: Diseño previo de arquitecturas mediante diagramas de clases estándar.
